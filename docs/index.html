オーストラリアの動物園について

昨日動物園に行ってきました．オーストラリアの動物園では多くの動物が放し飼いにされています．

特にエミューの放し飼いは怖かったです。足は恐竜みたいでしたが，普通に触れました。

カンガルーも放し飼いでしたし，コアラはユーカリを食べていて、意外とよく動いていました。

また、大きな蛇もいて迫力がありました。

全体的に楽しめたので、日本に帰ったら他の動物園にも行ってみようと思いました。
